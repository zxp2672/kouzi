## 迁移步骤

### 步骤 1：创建对应的服务（如需要）

参考已有的 `product-service.ts` 和 `warehouse-service.ts` 的模式，为每个需要的模块创建服务。

**服务文件模板：**

```typescript
import { getSupabaseClient } from '@/storage/database/supabase-client';

export interface YourModel {
  id: number;
  // 其他字段...
}

export interface YourModelFormData {
  // 表单字段...
}

const DEFAULT_DATA: YourModel[] = [
  // 默认数据...
];

async function isSupabaseAvailable(): Promise<boolean> {
  try {
    getSupabaseClient();
    return true;
  } catch {
    return false;
  }
}

export async function fetchYourModels(): Promise<YourModel[]> {
  const hasSupabase = await isSupabaseAvailable();
  
  if (!hasSupabase) {
    // Fallback to localStorage
    try {
      const saved = localStorage.getItem('your_storage_key');
      if (saved) {
        const parsed = JSON.parse(saved);
        return parsed.yourModels || DEFAULT_DATA;
      }
      localStorage.setItem('your_storage_key', JSON.stringify({ yourModels: DEFAULT_DATA, nextId: 2 }));
      return DEFAULT_DATA;
    } catch {
      return DEFAULT_DATA;
    }
  }

  try {
    const client = getSupabaseClient();
    const { data, error } = await client
      .from('your_table_name')
      .select('*')
      .order('created_at', { ascending: true });

    if (error) {
      console.warn('Supabase fetch failed, falling back to localStorage:', error);
      // Fallback to localStorage (同上)
    }

    return (data as YourModel[]) || [];
  } catch (error) {
    console.warn('Supabase not available, using localStorage:', error);
    // Fallback to localStorage (同上)
  }
}

// 类似地创建 create、update、delete 函数...
```

const fetchOrganizations = async () => {

    setLoading(true);

    // 构建树形结构
    const tree = buildOrganizationTree(savedData || []);

  }

## 通用模式

所有页面的迁移都遵循这个通用模式：

### 1. 导入服务

```typescript
import { ModelType, ModelFormType, fetchModels, createModel, updateModel, deleteModel } from '@/services/your-service';
```

### 2. 替换 fetch 函数

```typescript
// 之前：从 localStorage 读取
const savedData = getFromLocalStorage();

// 之后：从服务读取
const savedData = await fetchModels();
```

### 3. 替换保存/编辑

```typescript
// 之前：保存到 localStorage
saveToLocalStorage(updatedData);

// 之后：使用服务
if (editing) {
  await updateModel(id, formData);
} else {
  await createModel(formData);
}
```

### 4. 替换删除

```typescript
// 之前：从 localStorage 删除
const updated = allData.filter(item => item.id !== id);
saveToLocalStorage(updated);

// 之后：使用服务
await deleteModel(id);
```

## 注意事项

1. **ID 管理**：数据库会自动处理自增 ID，不再需要手动管理 `nextId`
2. **字段名**：确保数据库字段名是 snake_case（如 `created_at`）
3. **降级机制**：所有服务都包含完整的降级机制，即使 Supabase 不可用也能正常工作
4. **数据类型**：注意 numeric/decimal 等类型在 JavaScript 中的处理（使用字符串或 number）
